# Program Analisa Frekuensi Kata

Program ini adalah aplikasi CLI (Command Line Interface) sederhana untuk menyimpan kalimat, menampilkan daftar kalimat, dan menganalisis frekuensi kata dari kalimat yang telah disimpan. Setiap kalimat disimpan menggunakan judul unik, dan hasil analisa akan menampilkan jumlah kemunculan setiap kata serta total kata unik.

Program dibuat sepenuhnya dengan Python untuk melatih:
- manipulasi string,
- pembersihan tanda baca,
- penggunaan struktur data dictionary,
- serta pembuatan menu interaktif berbasis terminal.

---

## Fitur Utama

### 1. **Input Kalimat**
- Pengguna dapat memasukkan kalimat baru dengan judul unik.
- Judul tidak boleh kosong, tidak boleh lebih dari 18 karakter, dan tidak boleh duplikat.

### 2. **Tampilkan Kalimat**
- Menampilkan seluruh kalimat yang sudah tersimpan dalam format tabel.
- Otomatis melakukan wrapping jika kalimat terlalu panjang.

### 3. **Analisa Frekuensi Kata**
- Memilih kalimat berdasarkan judul untuk dianalisa.
- Menghapus tanda baca sebelum perhitungan.
- Menghitung jumlah kemunculan setiap kata.
- Menampilkan total kata unik.

### 4. **Quit**
- Menutup program.
